## Prerequisites

- npm >=1.22.4
- node >=12.18.1
- react >=16.13.1
- typescript >=3.7.2

## Install

```sh
yarn install
```

## Usage

```sh
yarn run start
```

## Run tests

```sh
yarn run test
```

## Using Docker

```sh
docker build -t codepen:1.0 ./
```

```sh
docker run -it --name codepen -p 3000:3000 -v $PWD:/usr/src/app codepen:1.0
```

## Using docker-compose

```sh
docker-compose up -d --build
```

```sh
docker-compose up
```
